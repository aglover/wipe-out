# Wipe Out!

A simple Node (ok, fine, CoffeeScript) script to remove users from App47 system. 

## Installation

```
$> npm install
```

Then set some environment variables, namely:

 * `APP47_API_TOKEN`
 * `APP47_URL`

And then:

```
$> coffee UserDelete.coffee <some Excel file>
```
